# AnimatedUI

This repository is a Unity project created as part of my personal portfolio.  
Its main purpose is to demonstrate practical skills in building and animating user interfaces using the Unity Engine and C#.

The project focuses on clean structure, reusable components, and smooth UI animations commonly used in games and interactive applications.  
It is distributed under the **MIT License**.

## Project Overview

This project showcases:

- UI animation techniques using Unity
- Organized and scalable project structure following Unity best practices
- Use of third-party UI and utility packages
- Clear separation between UI assets, logic, and configuration

## Technologies and Tools Used

- **Game Engine:** Unity
- **Programming Language:** C#
- **Version Control:** Git

## How to Download and Run

1. Clone the repository:

   ```bash
   git clone https://github.com/rafacf95/AnimatedUI.git
   ```

2. Access the project directory:

   ```bash
   cd AnimatedUI
   ```

3. Open the project in Unity Hub using the version corresponding to the repository.

4. Press **Play** to run the project.

## Use Case

This project is intended for:

- Portfolio demonstration
- Learning and experimentation with animated UI in Unity
- Reference for UI animation patterns in small to medium-sized projects
